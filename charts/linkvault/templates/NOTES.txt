=============================================================================
ACCESSING THE APPLICATION
=============================================================================

{{- if .Values.ingress.enabled }}

1. Get your Minikube IP:
   minikube ip

2. Add entry to your hosts file (C:\Windows\System32\drivers\etc\hosts):
   <minikube-ip> {{ (index .Values.ingress.hosts 0).host }}

3. Open your browser:
   http://{{ (index .Values.ingress.hosts 0).host }}

{{- else }}

Use port-forwarding to access the services:

  # Frontend
  kubectl port-forward -n {{ .Values.namespace }} svc/{{ .Values.frontend.name }} 4200:{{ .Values.frontend.service.port }}
  
  # Backend API
  kubectl port-forward -n {{ .Values.namespace }} svc/{{ .Values.backend.name }} 5000:{{ .Values.backend.service.port }}

Then open http://localhost:4200

{{- end }}
